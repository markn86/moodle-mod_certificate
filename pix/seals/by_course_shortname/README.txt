Please name the backgrounds in this folder as follows:

 COURSE_SHORTNAME_your_description.jpg

 or

 COURSE_SHORTNAME_your_description.png

If you don't have any description to provide then add something
generic such as _seal.jpg or _default.jpg.

Please not that malformed names are going to be ignored.

The seals stored in this folder that follow the naming
convention explained above are going to be visible only in 
courses with the corrispondent shortname.
