Please name the backgrounds in this folder as follows:

 COURSE_SHORTNAME_your_description.jpg

 or

 COURSE_SHORTNAME_your_description.png

If you don't have any description to provide then add something
generic such as _watermark.jpg or _default.jpg.

Please note that malformed names are going to be ignored.

The watermarks stored in this folder that follow the naming
convention explained above are going to be visible only in 
courses with the corrispondent shortname.

Please note that the filename length shouldn't be longer than 28 chars.
This is a limitation releated to the tables of this module.
If you modify the lenght of the tables please modify also
the FILE_NAME_LENGTH constant in the lib.php file.
